Twenty Fifteen Child Theme

A ready to modify bare-bones child theme for use with Twenty Fifteen Theme.

To use this Child Theme, you must have Twenty Fifteen parent theme installed.

For more information about Twenty Fifteen Child Theme please go to:
http://hyperlinkcode.com/blog/twenty-fifteen-child-theme/

For more information about Twenty Fifteen parent theme please go to:
https://wordpress.org/themes/twentyfifteen/

To find matching colors quickly please visit:
http://www.html-color-names.com

==
Installation
==


1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.


2. Click 'Upload Theme'.

3. Click 'Choose File' and browse to twentyfifteen-child.zip on your hard drive and select it.

4. Once you have selected twentyfifteen-child.zip, click 'Install Now'

5. Click on the 'Activate' button to use your new theme right away.



6. Navigate to Appearance > Editor in your WordPress admin panel to add custom modifications.
